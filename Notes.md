On String formats:

3 ways that I know so far.
 - f'thestring {variable1} {variable2}'
 - 'thestring %s %s' %(variable1, variable2)
 - '{} {}'.format(variable1, variable2)
 
 ````
var1 = 'foo'
var2 = 'bar'
print(f'this is {var1} {var2}')
print('this is %s %s' % (var1, var2))
print('this is {} {}'.format(var1, var2))
 ````
